rc4

在IDA中，
WORD代表一个字（1个字=2个字节=16位）
_DWORD代表两个字（2个字=4个字节=32位）
_QWORD（4个字=8个字节=64位）
_OWORD（8个字=16个字节=128位）
没有壳，直接看加密函数

这里把部分函数名改回来后，主要是rc4加密函数比较难看
传了六个变量，进去看函数很麻烦
其实v5v6用来生成S盒 后面传入input 并且跟着一个strlen
最后加了一个v7 值是moectf2023，后面跟了10，那大概率也就是v7长度了

    import requests

url = "http://localhost:53530"
session = requests.session()

data_res = ['弯道向左', '弯道向右', '弯道直行']
data_ym = ['抓地力太小了', '抓地力太大了', '保持']

    'driver': 'user',

}

for i in range(7):

    end_marker = '</div>'
    info = res_text.split(start_marker)[1].split(end_marker)[0]

    if data_res[0] in info:
        if data_ym[0] in info:
            fx = 1
            youmen = 0
        elif data_ym[1] in info:
            fx = 1
            youmen = 2
        elif data_ym[2] in info:
            fx = 1
            youmen = 1
    elif data_res[1] in info:
        if data_ym[0] in info:
            fx = -1
            youmen = 0
        elif data_ym[1] in info:
            fx = -1
            youmen = 2
        elif data_ym[2] in info:
            fx = -1
            youmen = 1
    elif data_res[2] in info:
        if data_ym[0] in info:
            fx = 0
            youmen = 0

            fx = 0
